# Habit Tracker

A simple static web app to help you build good habits, one day at a time.

Built with plain HTML, CSS, and JavaScript - no frameworks, no build step.
Just open `index.html` in a browser to run it.

## Project structure

- `index.html` - the main page
- `styles.css` - styling for the page
- `app.js` - the page's JavaScript

## Running it

Open `index.html` directly in any modern web browser. There is nothing to
install or compile.
